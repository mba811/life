# 借我一双慧眼，穿透浊世的迷雾

朋友在京效面见了一位姓赵的隐士，发出激动的感叹：“人的一切智慧，是从静定中来的”。类似的观点，宗教中很多“大师”都说过。我看了几篇隐士的文章，笑了 笑，这样的人，多一个不多，这样的著作，少一本不少，佛教本讲究“戒定慧”。说实话，我比较讨厌辩论宗教中的问题，因为通常需要一定的篇幅 才能讲清。也好，借这个机会谈谈我对智慧的一些看法。
